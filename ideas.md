## 아이디어 1: 테크니컬 블루프린트 (Technical Blueprint)

**디자인 무브먼트**: 스위스 국제 타이포그래피 스타일 + 테크니컬 다이어그램의 정밀함

**핵심 원칙**:
- 그리드 기반의 엄격한 정렬과 수학적 비율
- 기능이 형태를 결정하는 실용주의
- 불필요한 장식 제거, 정보 우선
- 계층적 타이포그래피로 명확한 정보 구조

**색상 철학**:
- 주조색: 진한 네이비 블루 (#1a2332) - 신뢰성과 전문성
- 강조색: 밝은 시안 (#00d9ff) - 기술적 정밀함
- 배경: 차가운 회색 톤 (#f5f7fa) - 중립적이고 깨끗한 캔버스
- 텍스트: 순수 검정과 회색 스케일 - 최대 가독성

**레이아웃 패러다임**:
- 고정 사이드바 (240px) + 유동적 콘텐츠 영역
- 8px 그리드 시스템 기반의 모든 간격
- 수평선과 수직선으로 명확한 섹션 구분
- 카드 기반이 아닌 연속적인 문서 흐름

**시그니처 요소**:
- 사이드바 메뉴 항목에 왼쪽 보더 인디케이터 (4px 시안 라인)
- 섹션 헤더에 기술 도면 스타일의 라벨 번호 (1.1, 1.2 등)
- 모노스페이스 폰트로 코드/기술 용어 표시

**인터랙션 철학**:
- 즉각적이고 예측 가능한 반응
- 호버 시 미묘한 배경색 변화 (5% 밝기 증가)
- 클릭 시 명확한 시각적 피드백 (active state)

**애니메이션**:
- 트랜지션은 150ms 이하로 빠르고 직선적
- 페이드 효과 없이 슬라이드와 스케일만 사용
- 사이드바 확장/축소는 cubic-bezier(0.4, 0, 0.2, 1)

**타이포그래피 시스템**:
- 헤딩: IBM Plex Sans (600-700 weight) - 기술적이고 명확
- 본문: Inter (400-500 weight) - 뛰어난 가독성
- 코드/라벨: JetBrains Mono - 기술 문서의 정확성
- 크기 스케일: 14px(본문), 16px(소제목), 24px(제목), 32px(페이지 타이틀)

## 아이디어 2: 모던 에디토리얼 (Modern Editorial)

**디자인 무브먼트**: 매거진 레이아웃 + 브루탈리즘의 대담함

**핵심 원칙**:
- 타이포그래피가 주인공인 콘텐츠 중심 디자인
- 비대칭 레이아웃으로 시각적 긴장감 생성
- 넓은 여백과 호흡하는 공간
- 대담한 타입 크기 대비로 계층 구조 강조

**색상 철학**:
- 주조색: 따뜻한 차콜 (#2d2d2d) - 에디토리얼의 무게감
- 강조색: 선명한 오렌지 (#ff6b35) - 에너지와 주목성
- 보조색: 올리브 그린 (#6a994e) - 자연스러운 균형
- 배경: 크림 화이트 (#fdfdf8) - 종이 같은 질감

**레이아웃 패러다임**:
- 비대칭 2단 그리드 (사이드바 280px, 콘텐츠 가변)
- 콘텐츠 영역 내 3단 타이포그래피 그리드
- 의도적인 정렬 깨기 (hanging punctuation, offset headings)
- 큰 타이틀은 사이드바를 넘어 콘텐츠 영역으로 확장

**시그니처 요소**:
- 초대형 섹션 번호 (200px 높이, 투명도 10%)
- 사이드바 메뉴 항목에 작은 컬러 도트 인디케이터
- 본문 내 풀아웃 인용구 (pull quotes) 스타일의 중요 정보

**인터랙션 철학**:
- 부드럽고 유려한 움직임
- 호버 시 요소가 살짝 확대 (scale: 1.02)
- 스크롤 시 사이드바 헤더가 sticky로 고정

**애니메이션**:
- 300-400ms의 여유로운 트랜지션
- ease-out 커브로 자연스러운 감속
- 페이지 전환 시 콘텐츠가 아래에서 위로 페이드인
- 사이드바 메뉴는 stagger 효과로 순차 등장

**타이포그래피 시스템**:
- 헤딩: Playfair Display (700-900 weight) - 에디토리얼 우아함
- 본문: Source Sans Pro (400-600 weight) - 편안한 독서 경험
- 라벨/캡션: DM Sans (500 weight) - 현대적 산세리프
- 크기 스케일: 16px(본문), 20px(소제목), 36px(제목), 56px(페이지 타이틀)

## 아이디어 3: 소프트 미니멀리즘 (Soft Minimalism)

**디자인 무브먼트**: 일본 미니멀리즘 + 스칸디나비아 디자인의 따뜻함

**핵심 원칙**:
- 부드러운 곡선과 라운드 코너로 친근함 표현
- 미묘한 그림자와 레이어로 깊이감 생성
- 여백을 통한 시각적 휴식 공간 확보
- 절제된 색상으로 집중력 유지

**색상 철학**:
- 주조색: 부드러운 인디고 (#4a5568) - 차분하고 전문적
- 강조색: 연한 민트 (#7dd3c0) - 신선하고 현대적
- 보조색: 따뜻한 베이지 (#e8dcc4) - 아늑함과 접근성
- 배경: 거의 흰색 (#fafbfc) - 공간감과 청결함

**레이아웃 패러다임**:
- 플로팅 사이드바 (카드 형태, 그림자 있음)
- 콘텐츠 영역은 최대 너비 제한 (720px)으로 가독성 최적화
- 섹션 간 넉넉한 수직 간격 (80-120px)
- 모든 컨테이너에 12-16px 라운드 코너

**시그니처 요소**:
- 사이드바 메뉴 활성 항목에 부드러운 배경 필 (민트색, 투명도 15%)
- 콘텐츠 카드에 미묘한 그림자 (0 2px 8px rgba(0,0,0,0.04))
- 검색창에 유리 모피즘(glassmorphism) 효과

**인터랙션 철학**:
- 부드럽고 예측 가능한 반응
- 호버 시 요소가 살짝 떠오르는 느낌 (그림자 증가)
- 클릭 시 ripple 효과로 피드백

**애니메이션**:
- 250ms의 균형잡힌 트랜지션
- ease-in-out 커브로 자연스러운 가속/감속
- 메뉴 확장 시 height와 opacity 동시 애니메이션
- 스크롤 시 요소들이 부드럽게 페이드인 (intersection observer)

**타이포그래피 시스템**:
- 헤딩: Outfit (500-700 weight) - 기하학적이고 현대적
- 본문: Noto Sans KR (400-500 weight) - 한글 최적화 가독성
- 라벨: Manrope (600 weight) - 둥근 형태의 친근함
- 크기 스케일: 15px(본문), 18px(소제목), 28px(제목), 40px(페이지 타이틀)

## 선택된 디자인: 소프트 미니멀리즘

내부 운영자용 매뉴얼이라는 특성상, 장시간 사용해도 눈이 편안하고 정보를 빠르게 찾을 수 있는 **소프트 미니멀리즘** 접근이 가장 적합합니다. 부드러운 색상과 넉넉한 여백으로 사용자 피로도를 줄이고, 명확한 계층 구조로 효율적인 정보 탐색을 지원합니다.
